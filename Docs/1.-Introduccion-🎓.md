## **1.1 Propósito**

El presente documento tiene como propósito describir de manera detallada los requisitos del sistema **Notiz (Java Version)**, una plataforma web de alto rendimiento y arquitectura moderna desarrollada para instituciones educativas. El objetivo primordial es facilitar la gestión académica mediante el registro de calificaciones por parte de los profesores, la consulta de notas por parte de los estudiantes, la generación de reportes académicos por parte de los administradores y la auditoría completa de los procesos.

Esta versión de la documentación define la especificación para la implementación basada en el ecosistema **Java 21 LTS**, **Spring Boot 3.x** y **MongoDB**, sustituyendo arquitecturas previas con un enfoque de alto rendimiento, escalabilidad horizontal, seguridad avanzada y resiliencia.

## **1.4 Alcance del Producto**

El sistema **Notiz (Java Version)** es una aplicación web empresarial multitenant diseñada para centralizar, automatizar y asegurar el registro, consulta y análisis de rendimiento académico en instituciones educativas. 

A diferencia de la especificación original, esta variante está optimizada para aprovechar el modelo de documentos de **MongoDB**, permitiendo una flexibilidad incomparable al gestionar escalas de calificación dinámicas, observaciones personalizadas e historiales de auditoría sin la rigidez de los esquemas relacionales.

El sistema proporcionará las siguientes capacidades principales a través de una API REST robusta:
* **Autenticación y Autorización Federada:** Soporte para **Google Identity (OAuth2 / Google SSO)** y autenticación local mediante **Spring Security + JWT stateless**, implementando control de acceso basado en roles (RBAC).
* **Gestión de Calificaciones Flexibles:** Registro de notas individuales, importación masiva desde plantillas Excel/CSV, configuración dinámica de criterios de evaluación de asignaturas y cálculo automático de promedios ponderados.
* **Motor de Reportes y Análisis:** Generación de estadísticas de rendimiento y exportación a documentos **PDF/A-1a** (cumpliendo con directivas de accesibilidad de largo plazo) y hojas de cálculo Excel.
* **Comunicaciones Asíncronas y Notificaciones:** Envío automatizado de correos mediante SendGrid API e interacción en tiempo real a través de **Spring WebSockets con protocolo STOMP**.
* **Auditoría Avanzada (Responsabilidad):** Trazabilidad de accesos y modificaciones persistidos de manera inmutable en MongoDB.

**Exclusiones:** El alcance de Notiz se limita estrictamente a la gestión académica, calificaciones, comunicación relacionada y reportes de rendimiento. Quedan excluidos los módulos financieros, gestión de nómina, control de asistencia presencial y administración de recursos físicos de la institución.
